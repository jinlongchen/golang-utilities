# golang-utilities


##  description
Some utilities for golang can improve development speed to some extent

## features:

   * alibaba/tencent/qiniu/baidu API
   * Compression library
   * Common type conversions
   * Encryption library
   * File library
   * Network library
   * JSON library
   * map library
   * Text operations
   *Excel operations

## How to install:
```bash
  go get github.com/jinlongchen/golang-utilities
```
